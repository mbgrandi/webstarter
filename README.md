# Webstarter
Java Web App Starter

Basic Orders/Products JPA schema, starting point for Java Web Development using:

- Spring Boot 2.2.4
- Thymeleaf 3.0.11
- Tomcat embedded 9.0.30
- JUnit 5.5.2
- Maven 3
- Java 8
- Bootstrap 4.4.1
- jQuery 3.4.1
- DataTables 1.10.20
- Chart.js 2.9.3
- H2 embedded database 1.4.2
- Jackson 2.10.2

Enjoy!
